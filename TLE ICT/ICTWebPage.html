HTML Formatting Elements

Bold: Makes text bold to emphasize importance.
Example: <b>This is bold text.</b>

Italic: Italicizes text, often used for emphasis or titles.
Example: <i>This is italicized text.</i>

Underline: Underlines text, sometimes used for links or emphasis.
Example: <u>This is underlined text.</u>

Delete: Draws a line through text, often used to indicate something is deleted or outdated.
Example: <del>This text is struck through.</del>

Highlight: Highlights text, drawing attention to it.
Example: <mark>This text is highlighted.</mark>

Small: Displays text in a smaller font size.
Example: <small>This is small text.</small>

Superscript: Raises text slightly above the baseline, often used in mathematical expressions or footnotes.
Example: E = mc<sup>2</sup>

Subscript: Lowers text slightly below the baseline, commonly used for chemical formulas.
Example: H<sub>2</sub>O

Image Maps

The <map> tag defines an image map. An image map is an image with clickable areas. The areas are defined with one or more <area> tags.

Areas

A clickable area is defined using an <area> element. Shape VALUES of the clickable area: rect - defines a rectangular region ,circle - defines a circular region ,poly - defines a polygonal region , and default - defines the entire region
